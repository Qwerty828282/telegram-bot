# Telegram Bot

Telegram-бот со сборками, туториалами (система ключей), поддержкой и админ-панелью.

## Возможности

- 📦 **Сборки** — публикация и просмотр сборок (текст + медиа)
- 📚 **Туториалы** — доступ по одноразовым ключам
- 🆘 **Поддержка** — приём обращений с кулдауном, ответы от админов
- 🔧 **Админ-панель** — управление контентом, ключами, пользователями

## Переменные окружения

| Переменная | Описание |
|---|---|
| `BOT_TOKEN` | Токен бота от @BotFather |
| `OWNER_ID` | Ваш Telegram user_id (число) |
| `DATABASE_URL` | PostgreSQL URL (обязательно на Railway) |

## Запуск локально

```bash
pip install -r requirements.txt
export BOT_TOKEN=your_token
export OWNER_ID=123456789
python bot/bot.py
```

## Деплой на Railway

1. Создайте проект на [railway.app](https://railway.app)
2. Добавьте PostgreSQL: **New → Database → PostgreSQL**
3. Задайте переменные `BOT_TOKEN` и `OWNER_ID`
4. `DATABASE_URL` подставится автоматически

## Команды

- `/start` — главное меню
- `/key XXXX-XXXX-XXXX-XXXX` — активация ключа доступа к туториалам
- `/admin` — открыть админ-панель (только для админов)
- `/dbinfo` — диагностика БД (только для админов)
- `/resetdb CONFIRM` — полный сброс БД (только для владельца)
